### 说明
一个地牢探险游戏，最终目的是到达最深处，拿到被埋葬的圣人永炼-千刃的传承。相比0.3版本，地图比较简单，改战棋模式为即时战斗模式。

### 操作
点击地面移动、点击敌人攻击。
